# 💏 PolyDating (연애 다항식) - Java Console Game

플레이어(여자)가 소개받은 남성 캐릭터를 선택하여
대화를 통해 관계를 발전시키는 **콘솔 기반 소개팅 시뮬레이션 RPG**입니다.
다양한 캐릭터와 데이트 코스를 선택하고, 돌발 상황에 대처하며 최선의 엔딩을 찾아보세요!

## 📌 주요 기능

### 1. 다양한 데이트 코스 (Choice 시스템)
5개의 패키지, 총 15개의 선택지 클래스가 준비되어 있습니다.
* **🏃 activity**: 번지점프, 실내 클라이밍, 한강 오리배
* **🍔 food**: 인스타 핫플 수제버거, 무한리필 삼겹살, 스시 오마카세
* **🎯 Hobby**: 코인 노래방, 고난이도 방탈출, 프라모델 조립
* **🏠 home**: 커플 랭크 게임, 배달 음식 마라탕/엽떡, 공포영화 정주행
* **✈️ travel**: 제주도 자전거 일주, 즉흥 강릉 바다 여행, 호텔 북캉스

### 2. 캐릭터별 맞춤형 반응
각 캐릭터(🏋️ 헬스남, 🎮 겜돌이, 😎 외향맨, 🌙 내향맨)는 성향에 따라 동일한 데이트 코스에도 각기 다른 반응과 점수 변화를 보입니다.

### 3. 랜덤 이벤트 시스템 (Event 시스템)
매 턴 30%의 확률로 돌발 상황이 발생합니다. 사용자의 대처(1~3번 선택)에 따라 호감도와 불쾌도가 변동됩니다.
* **주요 이벤트**: 갑자기 비가 오는 상황, 어색한 침묵, 친구와의 우연한 만남, 식사 예절, 계산 상황, 버스킹 공연, 갑자기 걸려온 전화, 강아지를 마주친 상황, 상대방이 넘어질뻔한 상황, 노을 감상 

### 4. 사용 메소드
* Event/Choice: event를 interface를 기반으로 생성 후, 이를 상속받아 대화를 진행
* Main
* 1. 생성한 event/choice를 각각 List<Map>에 담아 뽑아 사용할 수 있도록 함
* 2. count 10을 기준으로 while 게임루프를 돌림
* 3. Math.random()함수를 사용하여 event의 경우 30퍼센트의 확률로 발생할 수 있도록 함
* 4. choice에서 이미 선택한 주제는 목록에서 삭제하도록 설정함
* 5. getIntInput() 메소드로 Scanner 입력 시 숫자 범위 초과 및 문자 입력에 대한 예외처리를 구현함
* 6. checkEnding() 메소드로 매 턴 종료 후 호감도/불쾌도 조건을 체크하여 조기 엔딩 처리함
* 7. Character 추상 클래스를 상속받은 각 캐릭터가 react()를 오버라이딩하여 같은 선택지에도 캐릭터마다 다른 반응을 출력함 (다형성)

## 🏆 엔딩 조건
* **💖 연인 엔딩**: 호감도 90점 이상
* **🤝 친구 엔딩**: 호감도 60점 이상
* **💔 데이트 종료**: 불쾌도 70점 이상 (Game Over)
* **⏱️ TIME OUT**: 대화 횟수 10회 초과

## 🛠️ 실행 방법
1.  프로젝트를 Clone 합니다.
2.  `Application.java` 파일을 실행합니다.
